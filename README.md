# flutter_boilerplate

A scalable boilerplate for flutter apps.

## Features

- [x] A layered architecture
- [x] BLoC pattern
- [ ] Fluro router
- [ ] Themes
- [ ] Common Components
  - [ ] Button
  - [ ] Alerts
  - [ ] Progress Indicators
  - [ ] Bottom Sheets
- [ ] Fluid transitions between components
- [ ] JSON Serialization
